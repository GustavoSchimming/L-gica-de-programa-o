# 🚀 Log de Aprendizado: Lógica e Estruturação

## 📝 Descrição
Este repositório documenta a evolução da minha percepção analítica, focando na transição entre processos do mundo real e a lógica computacional.

---

## 🛠️ Tecnologias e Conceitos Aplicados

### 1. Pensamento Computacional
Apliquei a **Decomposição** para transformar uma tarefa cotidiana (fazer um bolo) em pequenas instruções lógicas e sequenciais.

### 2. Pseudocódigo
Utilizei uma linguagem intermediária para estruturar algoritmos sem a barreira de uma sintaxe rígida de programação.
* **Input (Entrada):** Ingredientes e utensílios.
* **Process (Processamento):** Mistura, ordem de inserção e tempo de espera.
* **Output (Saída):** O produto final (o bolo).
